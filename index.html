Welcome!

Welcome to Junior Game Crafters!

Originally started in 2015 as a responce to the lack of summer code camps in the Blanchardstown area.

We have spent a number of years now teaching children from all backgrounds about the joys of coding, computer science and game development, using tools available to everyone for FREE!

We believe that knowledge is something that should not be held only by those with money, that is why we always offer the summer camp completely for free, however if you want to donate we wont turn you down!
